# CalCity - California City Community Website

A modern, responsive web application for the California City community, showcasing local businesses, events, and visitor information.

![CalCity Website](https://images.unsplash.com/photo-1577303610524-3b00c10b1e65?w=500&auto=format&fit=crop&q=60&ixlib=rb-4.0.3&ixid=M3wxMjA3fDB8MHxzZWFyY2h8Nnx8bW9qYXZlJTIwY2F8ZW58MHx8MHx8fDA%3D)

## 🌵 About

CalCity is a comprehensive platform designed to connect residents and visitors with the California City community. The website provides information about local events, businesses, and serves as a guide for newcomers and tourists to this Mojave Desert gem.

## ✨ Features

- **Event Directory**: Browse and search upcoming community events
- **Business Listings**: Discover local businesses with detailed profiles
- **Visitor Guide**: Essential information for tourists and newcomers
- **Weather Widget**: Real-time weather updates for the area
- **Responsive Design**: Optimized for all devices from mobile to desktop
- **Dark Mode Support**: Comfortable viewing experience day and night

## 🚀 Technologies

- **React 18**: Modern UI library for building interactive interfaces
- **TypeScript**: Type-safe JavaScript for robust code
- **React Router**: Client-side routing for single-page application
- **Tailwind CSS**: Utility-first CSS framework for styling
- **Vite**: Next generation frontend tooling for fast development
- **Lucide Icons**: Beautiful, consistent icon set

## 🛠️ Getting Started

### Prerequisites

- Node.js (v18 or later recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/calcity.git
   cd calcity
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn
   ```

3. Start the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open your browser and navigate to `http://localhost:5173`

## 📦 Building for Production

```bash
npm run build
# or
yarn build
```

The built files will be in the `dist` directory.

## 🧪 Linting

```bash
npm run lint
```
